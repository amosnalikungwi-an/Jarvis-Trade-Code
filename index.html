MT5 bridge setup

To connect this app to your live EA, you need a small REST bridge running on your Windows VPS alongside MT5. Steps:

  1. Install Python on your VPS
  2. Run the MT5 Python bridge script (ask Claude to generate it)
  3. Open the port in Windows Firewall
  4. Enter your VPS IP:port above and tap Connect
